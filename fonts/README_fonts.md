# 字体文件说明

## 📁 字体文件夹说明

此文件夹用于存放字体文件，程序会自动从此文件夹加载所有可用的字体文件用于字符级随机替换。

## 🎯 字体要求

### 支持格式
- **TrueType 字体** (.ttf)
- **OpenType 字体** (.otf)

### 技术要求
- 必须包含英文字母和数字字符集
- 建议包含常用标点符号
- 支持 Unicode 编码

### 数量建议
- **最少**: 2 个字体文件（否则无法实现随机效果）
- **推荐**: 3-5 个字体文件（获得良好的随机效果）
- **最多**: 无限制（但过多可能影响性能）

## 📥 如何添加字体

### 方法一：手动添加
1. 将下载的 `.ttf` 或 `.otf` 文件复制到此文件夹
2. 重启应用程序
3. 程序会自动检测并加载新字体

### 方法二：使用程序功能
1. 在程序界面点击"打开字体文件夹"
2. 将字体文件复制到打开的文件夹中
3. 点击"刷新字体列表"

## ⚖️ 字体选择建议

### 风格搭配
- **相似风格**: 选择风格相近的字体，确保文档整体协调
- **对比风格**: 选择有明显差异的字体，增强随机效果
- **大小搭配**: 包含不同粗细和尺寸的字体

### 字符覆盖
- 确保字体包含完整的英文字母 (A-Z, a-z)
- 包含数字 (0-9)
- 包含常用标点符号 (.,!?;:'"等)

## ⚠️ 许可证重要提醒

### 字体版权
- **本程序不包含任何字体文件**，用户需自行提供
- 请确保您拥有所用字体的合法使用权
- 商业使用时请特别注意字体许可证

### 许可证类型
- **免费商用**: 可自由使用于商业项目
- **个人免费**: 仅限个人非商业使用
- **开源字体**: 遵循特定开源协议
- **付费字体**: 需要购买商业许可证

### 如何检查许可证
1. 下载字体时查看许可证说明
2. 检查字体文件属性中的许可证信息
3. 访问字体官方网站查看使用条款

## 🔧 故障排除

### 常见问题

**Q: 字体文件已添加但程序未识别**
A:
确保文件格式为 .ttf 或 .otf
检查文件是否损坏
点击"刷新字体列表"按钮
重启应用程序


**Q: 某些字符显示为方框**
A:
字体可能不包含该字符
尝试添加更多包含完整字符集的字体
程序会自动寻找支持该字符的字体


**Q: 字体加载失败**
A:
检查字体文件是否完整
尝试重新下载字体文件
某些字体可能有特殊加密或限制


### 字体文件检查清单

- [ ] 文件扩展名为 .ttf 或 .otf
- [ ] 文件大小合理（通常几百KB到几MB）
- [ ] 字体包含英文字母和数字
- [ ] 确认字体许可证允许使用
- [ ] 文件未被其他程序占用

## 📊 字体管理技巧

### 文件命名建议
使用清晰的命名方式，便于识别：
风格-字体名称.ttf
示例：
handwriting-caveat.ttf
script-dancing.ttf
casual-pacifico.ttf


### 性能考虑
- 字体文件越多，程序启动时间越长
- 建议使用 3-10 个字体文件获得最佳效果
- 单个字体文件不宜过大（建议小于 5MB）

### 备份建议
- 定期备份您收集的字体文件
- 创建字体清单记录许可证信息
- 保留字体来源链接

## 🌟 最佳实践

1. **测试字体效果**: 在添加前用少量文本测试字体显示效果
2. **混合字体类型**: 混合使用衬线体、无衬线体和手写体

4. **保持一致性**: 对于正式文档，选择风格相对统一的字体
